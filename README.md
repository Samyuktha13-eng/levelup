# Personalized Learning Roadmap Generator

A tool that generates step-by-step learning roadmaps for different career roles based on skill level.

## Features
- Career role selection (Software Developer, Data Scientist, DevOps Engineer)
- Skill level input (Beginner/Intermediate/Advanced)
- Step-by-step roadmap output
- Web interface and command-line options

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run web app: `python app.py`
3. Open browser to `http://localhost:5000`

## Command Line Usage
```bash
python roadmap_generator.py
```

## Web Interface
- Select career role from dropdown
- Choose your skill level
- Click "Generate My Roadmap"
- View personalized learning steps
